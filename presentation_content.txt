4. Selenium Kullanımı
Başlık: Selenium Kullanımı
İçerik:

IMDB'den Veri Çekme Süreci:
Sayfa gezinme ve veri toplama.
XPath ve CSS seçicilerle eleman bulma.
Kullanılan Yöntemler:
find_element, click, get_attribute.
Kod Örneği:
```python
from selenium import webdriver
from selenium.webdriver.common.by import By

# Selenium WebDriver başlatma
options = webdriver.ChromeOptions()
options.add_argument("--headless")
driver = webdriver.Chrome(options=options)

# IMDb sayfasına bağlanma
driver.get("https://www.imdb.com/chart/top")

# Başlıkları çekme
titles = driver.find_elements(By.CLASS_NAME, "titleColumn")
for title in titles[:5]:
    print(title.text)

driver.quit()
```

5. Veri Tabanı Yönetimi
Başlık: Veri Tabanı Yönetimi
İçerik:

MongoDB Kullanımı:
Bulut tabanlı ve ölçeklenebilir.
Kolay entegrasyon.
Veri Tabanı Şeması:
Koleksiyon yapısı ve alanlar:
id, title, rating, year.
Kod Örneği:
```python
from pymongo import MongoClient

class MongoDBManager:
    def __init__(self, uri, db_name, collection_name):
        self.client = MongoClient(uri)
        self.db = self.client[db_name]
        self.collection = self.db[collection_name]

    def insert_data(self, data):
        self.collection.insert_one(data)

# MongoDB bağlantısı
uri = "your_mongo_uri"
db_manager = MongoDBManager(uri, "IMDb_Archive", "Movies")

# Veri ekleme
data = {"title": "The Shawshank Redemption", "rating": 9.3, "year": 1994}
db_manager.insert_data(data)
```

6. İş Mantığı ve OOP
Başlık: İş Mantığı ve OOP
İçerik:

Kullanılan Sınıflar:
IMDbScraper, MongoDBManager.
OOP'nin Avantajları:
Kodun yeniden kullanılabilirliği.
Modüler yapı.
Kod Örneği:
```python
class IMDbContent:
    def __init__(self, title, rating, year):
        self.title = title
        self.rating = rating
        self.year = year

    def __str__(self):
        return f"{self.title} ({self.year}) - Rating: {self.rating}"

content = IMDbContent("The Godfather", 9.2, 1972)
print(content)
```
